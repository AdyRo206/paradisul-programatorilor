Primii pasi în dezvoltarea Web

  1. Introducere în dezvoltarea Web (4 ore)

    2. Prezentarea cursului și a limbajelor de programare abordate Introducere în dezvoltarea web și rolul fiecărui limbaj.

  2. HTML - Hypertext Markup Language (10 ore)

    3. Noțiuni de baza in HTML și explicații despre crearea și menținerea unui site, plus elemente de design ale unui website, cu exemplificare și ore de practica.

  3. CSS - Cascading Style Sheets (10 ore)

    4. Înțelegerea principiilor de bază ale stilizării paginilor web cu ajutorul CSS. Familiarizarea cu selecționarea elementelor și aplicarea stilurilor. Dezvoltarea abilităților de creare a unui design atractiv și responsive.

  4. JavaScript (8 ore)

    5. Înțelegerea fundamentelor limbajului de programare JavaScript. Aplicarea conceptelor de bază în manipularea documentului HTML și gestionarea evenimentelor. Dezvoltarea abilităților de programare și rezolvarea problemelor.

  5. Proiect Practic (4 ore)

    6. Dezvoltarea unui proiect web de baza pentru aplicarea cunoștințelor dobândite. Ghidare și feedback detaliat în timpul proiectului.

  • Suport Suplimentar:

    • Sesiuni avansate pentru rezolvarea întrebărilor și discutarea problemelor specifice. Forum online dedicat pentru interacțiunea între participanți și discuții avansate. Sesiuni intensive de întrebări și răspunsuri.

    Obiectivul cursului Obiectivul cursului - “Introducere în dezvoltarea Web” are ca scop introducerea studentilor în lumea limbajelor de programare, incepand cu HTML, CSS si JS. Cursul este destinat incepatorilor, și va include conceptele de baza ale limbajelor de programare menționate pentru a ajuta elevii interesați de a invata programare să-și înceapă studiile într-un mod cat mai clar și simplu.
    Materiale Didactice
  • Fiecare participant va primi materiale didactice detaliate, precum documentație, tutoriale și exemple de cod.
  • Teme săptămânale pentru a consolida cunoștințele dobândite.
  • Acces la o platformă online unde vor putea interacționa cu instrucțiunile, a trimite teme și a primi feedback.
    • Evaluare Evaluarea se va realiza prin intermediul proiectului complex și a unor teste avansate, oferind cursanților oportunitatea de a-și demonstra abilitățile la nivel înalt în dezvoltarea web.
